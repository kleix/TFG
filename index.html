Visualization of an HTML document in virtual reality

SUMMARY

This project presents a way to view HTML documents in virtual reality

Through the Data Object Model (DOM), which provides a tree-based structure with which to represent hierarchically the elements that make up the HTML page, we will be able to visualize each element that makes up the page in a simple way.

For 3D visualization, we have used A-Frame technology, a JavaScript framework, which is used to create scenes in virtual reality
